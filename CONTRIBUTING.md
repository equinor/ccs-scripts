## Contributing to ccs-scripts

Please feel free to reach out if you have any suggestions, questions, or bugs to report. It is possible to view the list of known tasks (bugs or new features) at [our issues page](https://github.com/equinor/ccs-scripts/issues).

We encourage new contributors to first discuss their ideas with us before beginning to write new code or submit pull requests.

Please note that each new pull request is subject to standard formatting checks (black, isort, mypy, flake8), as well as integration tests. Our stable branch is 'main,' while 'develop' may contain new, not yet fully tested features.
